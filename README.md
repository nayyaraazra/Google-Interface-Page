# CS50 Project 0: Google Interface Page

This project is a recreation of the Google Search interface using HTML and CSS. It includes the main search page, an image search page, and an advanced search page.

## Project Structure
* **`google_mainpage/`**: A replica of the classic Google Search landing page.
* **`google_imagesearch/`**: A replica of the Google Image Search interface.
* **`google_advanced_search/`**: A replica of the Google Advanced Search page, featuring various filters and input options.

## Features
* **Responsive Design**: The layouts aim to mimic the aesthetic and functionality of Google’s actual search interfaces.
* **Independent Styling**: Each page contains its own `style.css` file to handle specific layout and design requirements.
* **Navigation**: Users can easily navigate between the different search modes via clear links provided on each page.
